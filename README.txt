AI Study Helper (GPT-3.5)
==========================
Steps to Run:
1. Upload folder to Replit or open in VS Code.
2. Create a file named .env and add your key:
   OPENAI_API_KEY=your_openai_api_key_here
3. Click Run or use: python app.py
4. Open http://127.0.0.1:5000/ in browser.

Features:
✅ Ask AI to explain any topic
✅ Summarize long notes
✅ Generate 5-question quizzes
